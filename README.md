Loophole é um jogo orientado à estratégia e resolução de problemas,
no qual o jogador deve identificar oportunidades e explorar “brechas” dentro de cada cenário para progredir. 
A proposta combina raciocínio lógico, tomada de decisão e adaptação a diferentes desafios, proporcionando uma experiência dinâmica e envolvente.
O projeto busca não apenas entreter, mas também estimular habilidades cognitivas como análise, planejamento e criatividade.
